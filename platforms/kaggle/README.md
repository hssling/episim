# Kaggle Bundle

This folder contains the metadata needed to publish EPISIM demonstration assets to Kaggle.

## Intended Kaggle surfaces

- dataset: archived example runs and result tables
- notebook: executable overview of the EPISIM lab

## Typical workflow

1. Run EPISIM locally to generate example archives.
2. Copy the selected outputs into a Kaggle dataset directory.
3. Use the metadata files in this folder with the Kaggle CLI.
